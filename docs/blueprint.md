# **App Name**: iTest

## Core Features:

- Class and Student Management: Teachers can create new classes, invite students, and manage student accounts and access credentials within their classes.
- Assignment and Test Creation: Teachers can create various assignments, including short answer, long answer, multiple choice, and true/false questions, and assign them to specific classes or students.
- AI-Powered Document Digitization: Utilize an AI tool to convert uploaded PDF documents into editable text, enabling teachers to easily create assignments from existing materials.
- Student Submission Interface: Students can access assigned work, complete tasks using keyboard input, or a pen/drawing tool for annotations and illustrations, and submit their responses.
- Grading and Feedback System: Teachers can review submitted assignments, assign grades (1-5, accompanied by visual emoji indicators), and provide optional textual feedback to students.
- Secure Data Storage and Retrieval: A robust database solution for storing all class data, student information, assignments, submissions, and grades, with simple data sharing capabilities.

## Style Guidelines:

- Color scheme: Light mode. Primary accent color (deep and engaging blue): #295CA3. Background color (soft and neutral off-white): #EFF3F7. Complementary accent color (vibrant aqua): #2AC5C5.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, structured, and tech-forward feel. Body text font: 'Inter' (sans-serif) for high readability in longer content such as assignment descriptions and student responses.
- Use clear, minimalistic line-art icons that communicate function quickly, such as symbols for 'class', 'student', 'assignment', 'grade', and 'submit', to maintain a clean educational interface.
- Clean, structured, and responsive layouts ensuring content is accessible and well-organized on various devices (desktops, tablets, phones). Utilize clear segmentation for different sections like class lists, assignment details, and grading panels.
- Subtle, functional animations for feedback (e.g., assignment submission success, grade saving), navigation transitions, and showing progress, without being distracting.
